# Zeerostock Assignment Submission

This workspace contains both required parts of the Zeerostock assignment.

## Projects

- `inventory-search-project` -> Part A (Inventory Search API + UI)
- `inventory-database-project` -> Part B (Inventory Database + APIs)

## How to Run

### Part A

```bash
cd inventory-search-project/backend
npm install
npm start
```

Then open `inventory-search-project/frontend/index.html`.

### Part B

```bash
cd inventory-database-project
npm install
npm start
```
